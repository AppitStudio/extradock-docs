DockShare
=========

DockShare lets you prepare a dock layout for sharing with other ExtraDock users.

There are two related flows:

- **Share Dock** — Share one selected dock from the management window
- **Export for DockShare** — Create a share-safe package from Settings

Sharing a dock
--------------

Open the management window, select a dock, then use **Share Dock** in the dock header.

ExtraDock prepares a public preview of the dock and opens DockShare after the upload completes.

Share-safe data
---------------

DockShare packages are meant for public layouts. ExtraDock avoids including private local paths or sensitive widget data that should stay on your Mac.

For example, a public DockShare package should describe the layout and icons, not expose private folders or Stripe credentials.

Exporting a DockShare package
-----------------------------

Open **Settings** (⚙) → **Import / Export** → **Export for DockShare**.

This creates a ``.extradockshare`` file you can publish or send to someone else.

Importing shared docks
----------------------

Shared dock import is separate from full backup restore. Full backups are for your own complete setup; shared docks are for adding public dock layouts.

If you open an ExtraDock share link, ExtraDock asks for confirmation before importing the shared package.
