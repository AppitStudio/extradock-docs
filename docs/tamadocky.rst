TamaDocky
=========

TamaDocky is a virtual pet that lives in your dock.

Adding TamaDocky
----------------

1. Open the management window → select a dock → **Items**
2. Click **Add Widget**
3. Choose **TamaDocky**

Pet name
--------

Set a custom name for your pet in widget settings.

Widget size
-----------

Choose how much space TamaDocky has:

- **Regular** — Same as dock icon size
- **Medium** — Larger than a regular icon
- **Large** — Extra room for the pet

Pet mode
--------

Choose how you want to interact with TamaDocky:

- **Simple Pet** — A lightweight animated dock companion
- **Tamagotchi Mode** — Adds pet stats and care-style behavior

Tips
----

TamaDocky works best in docks with enough spacing or a larger icon size, especially in Medium or Large mode.
