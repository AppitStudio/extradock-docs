Background
==========

Each dock's background can use a blur material or a solid color, with adjustable opacity.

Open the management window → **Properties** → **Appearance** → **Background** to access these settings.

Background style
-----------------

Two styles are available:

- **Blur** — Frosted glass effect using macOS visual effect materials (default)
- **Solid** — Opaque background with a solid color

Blur intensity
--------------

When using the Blur style, five intensity levels are available:

- **Ultra Light**
- **Light**
- **Medium** (default)
- **Dark**
- **Ultra Dark**

Background color
----------------

When using the Solid style, choose a color from the built-in palette or use the custom color picker.

The color setting is disabled when the Blur style is active.

Opacity
-------

Controls overall background transparency. Three presets and a custom slider:

- **100%** — Fully opaque
- **85%** — Slight transparency
- **70%** — Noticeably translucent
- **Custom** — Slide to any value between 0% (invisible) and 100%

Default is 95%.
