IP Address Widget
=================

The IP Address widget shows your public IP address in the dock.

It fetches your IP from an external IP lookup service and refreshes automatically.

Using it
--------

Click the widget to refresh the IP address and copy it.

Display
-------

You can configure:

- **Show Text**
- **Show Icon**
- Text color
- Icon color
- Background color
- Font family
- Font size

Refresh interval
----------------

Choose how often the widget refreshes:

- 1 minute
- 2 minutes
- 3 minutes
- 5 minutes
- 10 minutes

The default refresh interval is 5 minutes.

Layout
------

The widget adapts to horizontal and vertical docks.
