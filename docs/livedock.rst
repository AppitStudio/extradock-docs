Live Dock
=========

Live Dock shows running apps or mirrors the native macOS Dock inside ExtraDock.

It is useful for dock replacement setups, task switcher docks, and multi-monitor workflows.

Modes
-----

Two modes are available:

- **Running Apps** — Shows apps that are currently running
- **Full Dock** — Mirrors the native macOS Dock contents

Running Apps mode is useful when you want a clean dock that only shows what is open right now.

Full Dock mode is useful when you want a second copy of the native Dock on another screen or edge.

Running indicators
------------------

Enable **Show running indicator dots** to show dots under running apps.

Hidden items
------------

Right-click an item in Live Dock to hide it from the widget. Open the widget settings to show hidden items again.

Accessibility
-------------

Full Dock mode can use Accessibility permission to better match the native Dock's exact order. If permission is missing, Live Dock still works with fallback behavior.
