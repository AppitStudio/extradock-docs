Space Awareness
===============

Space Awareness shows apps that have windows in the current macOS Space.

It updates as you switch Spaces, so you can see what belongs to the desktop you are currently using.

Permission
----------

Space Awareness requires Accessibility permission.

Without Accessibility, ExtraDock cannot detect apps in the current Space or control their windows.

Left-click action
-----------------

Choose what clicking an app does:

- **Minimize / Restore Window** — Toggle the app window state
- **Bring Window to Front** — Focus the app's window

Display
-------

You can configure:

- **Icon Spacing**
- **Show minimized app overlay** — Dim minimized apps so they stand out

Hidden items
------------

Right-click an item in the widget to hide it. Open widget settings to show hidden items again.

When to use it
--------------

Use Space Awareness when you rely on multiple macOS Spaces and want a quick view of the apps that belong to the current Space.
