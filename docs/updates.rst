Updates
=======

ExtraDock checks for updates automatically using the Sparkle framework.

Open **Settings** (⚙) → **Updates** to manage update preferences.

Checking for updates
--------------------

Click **Check for Updates** to manually look for a new version. You can also check from the menu bar tray icon.

When an update is available, a dialog shows the release notes and lets you download and install it in place.

Automatic updates
-----------------

By default ExtraDock periodically checks for updates in the background. You can adjust the frequency or disable automatic checks from the Updates settings page.
