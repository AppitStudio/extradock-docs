License
=======

Manage your ExtraDock license key and activation status.

Open **Settings** (⚙) → **License** to access these settings.

Activating a license
--------------------

1. Enter your license key in the text field

2. Click **Activate License**

3. ExtraDock verifies the key with the license server

On success, the status shows **Active** with a green checkmark.

License status
--------------

- **Active** — License is valid and verified
- **Not Activated** — No license key has been entered yet
- **Invalid** — The key was not recognized
- **Wrong Device** — The key is registered to a different machine
- **Server Error** — Could not reach the license server; try again later

Removing a license
------------------

Click **Remove License** to deactivate this device. You'll need to re-enter and activate a key to continue using ExtraDock.
