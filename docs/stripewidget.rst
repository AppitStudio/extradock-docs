Stripe Balance Widget
=====================

The Stripe Balance widget shows Stripe revenue data in your dock.

It is designed for developers and business owners who want a quick glance at Stripe without opening the dashboard.

Setup
-----

1. Open **Settings** (⚙) → **Integrations**
2. Save a Stripe API key
3. Add **Stripe Balance** from the widget picker

See :doc:`integrations` for key setup.

Data
----

Choose what the widget shows:

- **Gross** or **Net** volume
- Date ranges such as today, this week, this month, or custom dates

The widget fetches completed Stripe Checkout Sessions and caches data locally so it does not refetch more than it needs.

Refresh
-------

Use **Refresh Now** in widget settings to fetch the latest data.

You can also choose a refresh interval:

- 5 minutes
- 10 minutes
- 30 minutes
- 1 hour

Display
-------

Customize:

- Text and icon visibility
- Text color
- Icon color
- Background color
- Font family
- Font size

Click action
------------

Choose what happens when you click the widget:

- **No Action**
- **Open Stripe Dashboard**
- **Open Widget Settings**
