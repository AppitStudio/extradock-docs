Border
======

Add a border around a dock for visual definition.

Open the management window → **Properties** → **Appearance** → **Border** to access these settings.

Enable/disable
--------------

Toggle the border on or off. Borders are **on by default**.

Border width
------------

Control the thickness:

- **Hairline** — 0.5px
- **Thin** — 1.0px (default)
- **Bold** — 2.0px
- **Custom** — Slide to any value between 0.5px and 10px

Border color
------------

Choose from presets:

- **Clear** — No visible border color
- **Soft White** — White at 20% opacity
- **Bright White** — White at 50% opacity
- **Dark** — Black at 25% opacity

A custom color picker is also available.
