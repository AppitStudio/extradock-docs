Widgets
=======

Widgets are special dock items that go beyond app shortcuts. They add functionality like clocks, file shelves, URL bookmarks, and more directly into your dock.

Adding a widget
---------------

1. Open the management window → select a dock → **Items** tab

2. Click **Add Widget**

3. Choose a widget from the widget picker

Each widget has its own settings — click the gear icon on a widget in the Items tab to configure it.

Available widgets
-----------------

Layout widgets:

- :doc:`Spacer <spacerwidget>` — Add flexible space between items
- :doc:`Divider <dividerwidget>` — Add a visible separator line

Shortcuts and file widgets:

- :doc:`URL <urlwidget>` — Open websites, links, and URL schemes
- :doc:`Finder <finderwidget>` — Open common Finder locations
- :doc:`Folder Stack <folderstack>` — Keep multiple folders behind one icon
- :doc:`Trash <trashwidget>` — Use ExtraDock Trash or native macOS Trash
- :doc:`Shelf <shelf>` — Store temporary files in your dock

Status and utility widgets:

- :doc:`Clock <clockwidget>` — Show time, date, and timezone
- :doc:`IP Address <ipwidget>` — Show and copy your public IP address
- :doc:`Stripe Balance <stripewidget>` — Show Stripe balance data

Dock replacement widgets:

- :doc:`Live Dock <livedock>` — Show running apps or mirror the macOS Dock
- :doc:`Space Awareness <spaceawareness>` — Show apps with windows in the current Space

Fun widgets:

- :doc:`TamaDocky <tamadocky>` — Add a virtual pet to your dock

Configuring widgets
-------------------

Open the management window → select a dock → **Items** tab. Click the gear icon beside a widget to open its settings.

Right-clicking a widget in the dock may also show actions such as opening its settings, refreshing data, hiding an item, or removing the widget.
