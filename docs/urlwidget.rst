URL Widget
==========

The URL widget opens a website, link, or URL scheme from your dock.

You can use it for websites, mail links, app deep links, dashboards, documentation, or anything macOS can open as a URL.

Adding a URL widget
-------------------

1. Open the management window → select a dock → **Items**
2. Click **Add Widget**
3. Choose **URL**
4. Enter the URL in widget settings

If you enter a website without ``https://``, ExtraDock adds it automatically.

Title
-----

Enable **Show Title Below Icon** to show a label under the icon.

Leave the title empty to use the website domain, or enter a custom title.

Icon source
-----------

Choose how the widget icon appears:

- **Website Favicon** — Fetch the site's favicon automatically
- **SF Symbol** — Pick a built-in macOS symbol
- **Icon from URL** — Download an image from a URL
- **Icon from File** — Import PNG, JPG, or ICNS from your Mac

Colors
------

When using an SF Symbol, you can set the icon and background colors.

Quick presets include Blue, Green, Red, Orange, Purple, Light, and Dark.

Actions
-------

Click the widget to open the URL.

Right-click it for actions such as opening or copying the URL.
