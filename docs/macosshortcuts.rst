macOS Shortcuts & Focus
=======================

ExtraDock supports macOS Shortcuts and Focus Filters in addition to per-dock hotkeys.

App Shortcuts
-------------

ExtraDock provides these actions to the Shortcuts app and Spotlight:

- **Show Dock**
- **Hide Dock**
- **Toggle Dock**
- **List Docks**

Use these when you want ExtraDock to be part of a larger automation, such as changing your workspace, starting a meeting setup, or showing a specific dock before opening an app.

Focus Filters
-------------

ExtraDock can show or hide selected docks when a macOS Focus mode turns on.

In macOS Focus settings, add the ExtraDock filter and choose:

- **Docks to Show** — Docks that should become visible when the Focus activates
- **Docks to Hide** — Docks that should be hidden when the Focus activates

This is useful for separating work, personal, creative, and presentation layouts.

Hotkeys vs Shortcuts
--------------------

Use :doc:`hotkeys` when you want a direct keyboard shortcut for one dock.

Use macOS Shortcuts or Focus Filters when you want ExtraDock to react as part of a broader system automation.
