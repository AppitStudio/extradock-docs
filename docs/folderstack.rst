Folder Stack
============

Folder Stack keeps one or more folders behind a single dock icon.

Use it when you want quick access to a group of folders without filling the dock with separate folder icons.

Adding folders
--------------

1. Add a **Folder Stack** widget
2. Open its settings
3. Add one or more source folders

The widget shows the selected folders together under one icon.

Display mode
------------

Choose how folders appear when you open the stack:

- **List** — A compact vertical list
- **Grid** — A visual grid

Appearance
----------

You can customize:

- **Visible name** — Show a label under the widget icon
- **Tint color** — Apply a folder icon tint
- **Custom icon** — Use an icon from your custom icon library

Open folders with
-----------------

By default, folders open in Finder. You can choose another app, such as a file manager, in **Open Folders With**.
