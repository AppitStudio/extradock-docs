Trash Widget
============

The Trash widget adds trash functionality to any dock.

It can use ExtraDock's own virtual trash or the native macOS Trash.

Modes
-----

Two modes are available:

- **ExtraDock Trash** — A virtual trash managed by ExtraDock
- **Native Trash** — Uses the system macOS Trash

ExtraDock Trash
---------------

Drag files onto the widget to move them into ExtraDock Trash. Click the widget to browse trashed items.

Native Trash
------------

Native Trash works with the system Trash. macOS may ask for file permissions depending on where the files come from.

Settings
--------

You can configure:

- **Show badge** — Show the item count
- **Badge color** — Change the badge color
- **Icon style** — Auto, full, or empty
- **Confirm empty** — Ask before emptying trash
- **Confirm protected** — Ask before removing protected items
