Privacy & Diagnostics
=====================

ExtraDock keeps your dock configuration local unless you choose a feature that sends data elsewhere.

Error reports
-------------

Open **Settings** (⚙) → **General** → **Send error reports**.

When enabled, ExtraDock can send anonymous error reports after something goes wrong. These reports help diagnose crashes and stability issues.

You can turn this off at any time.

Backups
-------

Local backups are stored on your Mac. Exported ``.extradockbackup`` files are yours to store, move, or delete.

DockShare
---------

DockShare packages are designed for public sharing. ExtraDock prepares a share-safe format so private paths and sensitive widget data are not published.

Use :doc:`importexport` for personal backups and :doc:`dockshare` for public sharing.

Stripe
------

Stripe API keys are stored securely in your macOS Keychain. They are used by the Stripe Balance widget to fetch Stripe data directly from Stripe.
