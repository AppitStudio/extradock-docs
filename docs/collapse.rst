Collapse Mode
=============

Collapse mode lets you shrink a dock down to a single button. Click or hover to expand it back.

Setting a collapse direction
-----------------------------

- **Right-click** the dock → **Collapse Mode** → choose a direction
- **Management window** → **Properties** → **Behavior** → **Collapse** → **Mode**

Five options are available:

- **None** — No collapse (default)
- **Left to Right** — Button on the left, content expands rightward
- **Right to Left** — Button on the right, content expands leftward
- **Top to Bottom** — Button on top, content expands downward
- **Bottom to Top** — Button on bottom, content expands upward

When a direction is set, a collapse button appears at the edge of the dock. Click it to toggle between collapsed and expanded.

Hover to expand
---------------

When enabled, hovering over a collapsed dock automatically expands it. Moving away collapses it again.

- **Right-click** the dock → **Enable Hover-to-Expand**
- **Management window** → **Properties** → **Behavior** → **Hover to Expand**

Collapse button customization
------------------------------

You can customize the collapse button's icon. Open **Management window** → **Properties** → **Behavior** → **Collapse Button** to access these settings.

Four icon types are available:

- **Arrow (Default)** — Directional chevron that changes based on collapse state
- **SF Symbol** — Choose from 4000+ SF Symbols using the built-in picker
- **Text/Emoji** — Enter any single character or emoji
- **None** — No icon, just the button background

For SF Symbol and Text types, you can also set a custom icon color.
