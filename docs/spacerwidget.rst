Spacer Widget
=============

The Spacer widget adds empty space between dock items.

Use it to separate groups, push items apart, or make a dock feel less crowded.

Adding a spacer
---------------

1. Open the management window → select a dock → **Items**
2. Click **Add Widget**
3. Choose **Spacer**

Size
----

Spacer size can scale with the dock or use a custom value:

- **Quarter**
- **Half**
- **Single**
- **Double**
- **Custom**

Use **Custom** when you want an exact pixel width or height.

Orientation
-----------

In a horizontal dock, the spacer adds width. In a vertical dock, it adds height.
