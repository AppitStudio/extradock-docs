Fixed Mode
==========

Fixed mode pins a dock to a screen edge. The dock stays anchored to that edge and its position is calculated automatically.

Setting up
----------

- **Right-click** the dock → **Position Mode** → choose an edge (Left, Right, Top, Bottom)
- **Management window** → **Properties** → **Position** → **Mode** → **Fixed**, then pick a **Pinned Edge**

Pinned edges
------------

Four edges are available:

- **Left** — Vertical dock pinned to the left edge
- **Right** — Vertical dock pinned to the right edge
- **Top** — Horizontal dock pinned to the top edge
- **Bottom** — Horizontal dock pinned to the bottom edge

The management window shows an interactive edge picker for quick selection.

Full width
----------

When enabled, a fixed dock stretches to fill the full width (or height) of the screen.

**Management window** → **Properties** → **Position** → **Full Width**

This setting only applies to fixed mode docks.

Space from edges
----------------

Adds a small gap (5px) between the dock and the screen edge, matching the macOS Dock style.

**Management window** → **Properties** → **Position** → **Space from Edges**
